Introdução ao Visual Studio Code

[início] | [índice]

Baixe gratuitamente o VS Code. Ou através do site da Microsoft.com.

Após fazer o download, confira todos os passos da instalação e abra o VS Code seguindo esta introdução até concluir o projeto.

  • Na tela de boas-vindas, clique em "novo arquivo" e selecione "arquivo de texto";
  • Com a nova aba aberta, utilize as "tags" de acordo com as necessidades de estruturação do documento em HTML;
  • Salve o projeto a medida que inserir as tags e conteúdos do site, pressione as teclas [CTRL]+[S], que formam o atalho para salvar imediatamente o HTML.Fazendo isso pela primeira vez, abrirá uma janela do explorador de arquivos;
  • Escolha uma pasta, nomeie o arquivo como "index", selecione o tipo HTML e clique em salvar;
  • Pressione [F5] para abrir o projeto no navegador, assim poderá atualizar pressionando [CTRL]+[S];
  • A cada passo do projeto é possível conferir os resultados obitdos e possíveis correções a fazer.

Entenda a Estrutura do HTML

[início] | [intro vs code]

Primeira tag inserida é <html>, que define o tipo do documento que utilizaremos e que será interpretado pelo navegador para exibir todo conteúdo;

Segunda tag ficará entre abertura e fechamento de <html>, será <head> que recebe <title>

E por fim, a tag <body>, entre fechamento </head> e </html>

O esquema abaixo explica melhor a ordenação de cada tag na estrutura base:

  1. <html>
  2. <head>
  3. <title>
    • Título da página que aparecerá na aba do navegador
  4. </title>
  5. </head>
  6. <body>
    • Tags aplicadas ao conteúdo que será interpretado e ficará visível no navegador
  7. </body>
  8. </html>

As tags que vamos usar

[início] | [estrutura html]

Sugestão de uso e breve definição de cada tag.

  • <html> </html>: Container que receberá todo conteúdo visível e não visível do site;
  • <head> </head>: É a cabeça da estrutura, onde serão acrescentadas outras instruções, como a tag seguir;
  • <title> </title>: intitular a página, sendo o título exibida na parte superior na aba do navegador;
  • <body> </body>: está é o corpo da página, recebe todos os elementos que serão interpretados pelo navegador para exibir as informações no site;
  • <h1> </h1> até <h6> </h6>: Título - Sua ordem de importância inicia por "h1" que é título principal e utilizado uma vez. Sendo as demais para subtítulos.
  • <p> </p>: Parágrafo. Usada para descrever textos em geral. Para cada linha do texto, deve ser utilizada novamente;
  • <mark> </mark>: Marcador de texto. Marca uma ou a sequência de palavras envolvidas.;
  • <strong> </strong>: Negrito. Reforça a expessura da font;
  • <ol> </ol>: Lista ordenada. Esta tag determina que cada item da lista será ordenado por sequência numérica;
  • <ul> </ul>: Lista não ordenada. Os itens da lista são sinalizados com um "bullets" no seu início. Utilizado para qual não tem uma ordem para os itens;
  • <li> </li>: Item de lista. Serve para descrever os itens de uma lista;
  • <a href=""></a>: Âncoragem. Esta tag possui o atributo "href=" para redirecionamento para outra página dentro e fora do site, ou algum trecho identificado da página. tags que estamos usando. Google.com
  • <hr>: Linha horizontal. Esta não possuí a tag de fechamento. Normalmente usada para separa um trecho específico de texto;
  • <sub> </sub>: Subscrito. Normalmente usado para fórmulas quimicas. O texto fica no meio da liha do parágrafo. Exemplo: "H2O".
  • <sup> </sup>: Sobrescrito. Esta já é usada em fórmulas matemáticas, O texto fica acima da linha do texto do mesmo parágrafo. Por exemplo: "23= 8";
  • <blockquote> </blockquote>: Citação. Usada para quando temos como fonte outro autor
    Lorem, ipsum dolor sit amet consectetur adipisicing elit. Libero labore explicabo cupiditate aperiam excepturi. Accusamus, minus saepe? Deserunt esse necessitatibus quam, sapiente sint, tempora nemo est natus, earum nobis veritatis?
  • <font> </font>: Estilizar a font. Não é compativel com HTML5. Era utilizada no HTML4 para especificar o tipo, tamanho e a cor do texto;
  • <del> </del>: Indica troca de informação. Mostra o trecho tachado. A substituição é feita usando a tag <ins></ins> para mostrar uma correção. Por Exemplo: "O Geraldo, é funcionário ex-funcionário da empresa".
  • <abbr> </abbr>: Abreviatura. Usado para exibir o significado de uma sigla ou acrônimo numa caixa de texto flutuante que aparece quando apontada pelo ponteiro do mouse. Exemplo: HTML;
